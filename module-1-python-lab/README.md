# Module 1 Python Lab - Pizza Order Cost Calculator

A CLI program that calculates the total cost of a pizza order based on size, toppings, and delivery distance.

## How to Run

```bash
python pizza_order_cost.py
```

Follow the prompts to enter your pizza size, number of toppings, and delivery distance.
