# Module 7 - Task Management System

A CLI program for managing tasks with support for adding, completing, and tracking progress against due dates.

## Run

```bash
python main.py
```

## Structure

```
├── main.py
└── task_manager/
    ├── __init__.py
    ├── task_utils.py   ← task operations (add, complete, view, progress)
    └── validation.py   ← input validation (title, description, due date)
```
